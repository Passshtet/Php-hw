# Инструкция по запуску 

## 1. Установка зависимостей
Перейдите в папку проекта и установите пакеты:
```bash
composer install
npm install
```

## 2. Настройка окружения
Скопируйте конфигурационный файл:
```bash
cp .env.example .env
```
Сгенерируйте ключ приложения:
```bash
php artisan key:generate
```

## 3. База данных
Для работы приложения нужна база данных. По умолчанию используется SQLite.
Создайте пустой файл базы данных:
```bash
touch database/database.sqlite
```
Затем запустите миграции 
```bash
php artisan migrate 
```

## 4. Сборка фронтенда
Скомпилируйте статические файлы:
```bash
npm run build
```

## 5. Запуск
Запустите локальный PHP сервер:
```bash
php artisan serve
```

Приложение доступно по адресу: http://localhost:8000
