# Инструкция по запуску

## 1. Установка зависимостей
Для начала работы установите библиотеки PHP и Node.js:
```bash
composer install
npm install
```

## 2. Настройка окружения
Скопируйте файл настроек:
```bash
cp .env.example .env
```
Затем сгенерируйте уникальный ключ приложения:
```bash
php artisan key:generate
```

## 3. Сборка фронтенда
Скомпилируйте CSS и JS файлы (если требуется):
```bash
npm run build
```

## 4. Запуск сервера
Запустите локальный сервер разработки:
```bash
php artisan serve
```

Приложение доступно по адресу: http://localhost:8000
